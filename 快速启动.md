# 🚀 快速启动指南

## 一、环境准备（只需做一次）

### Windows 用户
1. 确保已安装 Python 3.9 ~ 3.12（推荐 3.10 或 3.11）
   - 下载地址：https://www.python.org/downloads/
   - 安装时**务必勾选** "Add Python to PATH"

2. 验证安装：打开 CMD，输入
   ```
   python --version
   ```
   显示版本号即为成功。

### Mac 用户
Mac 一般自带 Python，打开终端输入 `python3 --version` 确认。

---

## 二、安装依赖（只需做一次）

### Windows
在项目文件夹内，地址栏输入 `cmd` 回车，然后执行：
```
pip install streamlit pandas
```

### Mac
打开终端，cd 到项目目录，执行：
```
pip3 install streamlit pandas
```

⏳ 等待安装完成，大约 2-5 分钟。

---

## 三、启动应用

### 方法一：命令行启动（推荐）

在项目根目录下执行：

**Windows:**
```
cd app
streamlit run Home.py
```

**Mac:**
```
cd app
streamlit run Home.py
```

浏览器会自动打开页面，地址一般是 `http://localhost:8501`

### 方法二：双击启动（Windows）

双击 `启动.bat` 文件（我已帮你放好了）

---

## 四、常见问题

### ❌ 提示 "streamlit 不是内部或外部命令"
说明没安装成功，重新执行 `pip install streamlit`

### ❌ 页面打开空白 / 报错
1. 确认在 `app` 目录下执行的命令
2. 看看 CMD 里的红色错误信息，截图发我

### ❌ 中文显示乱码
这是 Streamlit 正常现象，不影响使用

### ❌ 端口被占用
换个端口启动：
```
streamlit run Home.py --server.port 8502
```
