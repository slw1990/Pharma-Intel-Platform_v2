# 💊 医药行业情报平台

> Pharma Intelligence Platform - AI驱动的医药行业数据库与分析工具

## 📋 项目简介

一个基于Streamlit的交互式医药行业情报平台，覆盖20家代表性中国上市药企，提供公司详情、财务对比、管线追踪、新闻动态等功能。

**MVP版本特性：**
- 🏢 **20家**代表性上市药企（创新药/CXO/中药/生物制品/医疗器械）
- 📊 **8个季度**财务数据（营收、净利润、研发投入、销售费用等）
- 💊 **200+条**管线数据（上市产品+在研管线）
- 📰 **行业新闻**动态追踪
- ⚖️ **多维度公司对比**（财务、管线、靶点等）
- 🔍 **管线筛选**与分析

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动应用

```bash
cd app
streamlit run Home.py
```

应用将在 `http://localhost:8501` 启动。

## 🎯 功能模块

### 1. 行业总览 Dashboard
- 20家企业整体市场概览（总市值、PE、涨跌家数）
- 分赛道分布统计
- 企业行情一览表
- 研发投入排行榜

### 2. 公司详情
- 公司基本信息与行情卡片
- 财务分析（营收/利润/研发/销售费用趋势图）
- 年度/季度财务对比表
- 管线布局统计与详情
- 按研发阶段筛选

### 3. 公司对比
- 基本信息对比
- 财务指标对比（2025年报）
- 营收/净利润趋势对比
- 管线数量与结构对比
- 靶点/技术平台对比

### 4. 管线追踪
- 全行业管线筛选（企业/阶段/类型/靶点）
- 研发阶段分布图
- 类型分布图
- 热门靶点Top20
- 管线明细表

### 5. 新闻动态
- 行业新闻列表（研发进展/经营动态/政策法规/并购整合）
- 多维度筛选（分类/企业/时间）
- 新闻分类统计
- 每日新闻趋势

## 🔄 数据更新

### 更新财务数据

```bash
cd scripts
python fetch_financial_light.py
```

> 注意：需要能访问东方财富等财经网站的网络环境。

### 更新管线数据

管线数据目前为手工整理，后续可接入：
- CDE药品审评中心数据
- 企业年报/公告
- 临床试验登记平台

### 更新新闻数据

新闻数据目前为示例数据，后续可接入：
- 巨潮资讯（企业公告）
- CDE/NMPA官网（政策法规）
- 医药魔方、药渡等行业媒体
- RSS订阅源

## 🛠 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Streamlit | 快速搭建交互式Web应用 |
| 数据处理 | Pandas | 数据清洗与分析 |
| 数据存储 | JSON文件 | MVP阶段轻量存储 |
| 数据采集 | Requests + BeautifulSoup | 网页爬虫与API调用 |
| 可视化 | Streamlit Charts | 内置图表组件 |
