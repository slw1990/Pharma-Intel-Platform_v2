# 网页版部署指南（Streamlit Community Cloud 免费版）

部署后你会得到一个公网链接，比如 `https://your-app.streamlit.app`，任何人都能在浏览器里打开，不用装任何软件。

---

## 准备工作

### 1. 注册 GitHub 账号（免费）
- 打开 https://github.com
- 点击 "Sign up" 注册一个账号
- 记住你的用户名和密码

### 2. 注册 Streamlit Community Cloud（免费）
- 打开 https://streamlit.io/cloud
- 点击 "Sign up" → 选择 "Continue with GitHub"
- 用你刚才的 GitHub 账号授权登录

---

## 部署步骤（一共 5 步）

### 第 1 步：在 GitHub 创建一个新仓库

1. 登录 GitHub 后，点击右上角的 `+` → 选 `New repository`
2. 填写：
   - **Repository name**：填 `pharma-intel-platform`（随便起，英文就行）
   - **Public** 选上（公开的才能免费部署）
   - 勾选 `Add a README file`
3. 点击 `Create repository`

### 第 2 步：上传代码文件到 GitHub

有两种方式，选一种就行：

#### 方式 A：网页直接上传（最简单，推荐）

1. 进入你刚创建的仓库页面
2. 点击 `Add file` → `Upload files`
3. 把你电脑里 `医药情报平台MVP_v3` 文件夹里的**所有文件和文件夹**都拖进去
   - 包括 `app/`、`data/`、`scripts/`、`requirements.txt`、`README.md` 等
   - 注意：要把整个文件夹的内容上传，不是上传 zip 包
4. 拉到页面底部，点击 `Commit changes`

#### 方式 B：用 Git 命令行（适合后续频繁更新）

如果你以后想经常更新代码，建议装一下 Git。需要的话告诉我，我给你更详细的步骤。

### 第 3 步：在 Streamlit Cloud 部署

1. 登录 https://share.streamlit.io
2. 点击右上角的 `New app`
3. 填写：
   - **Repository**：选择你刚才创建的仓库（比如 `你的用户名/pharma-intel-platform`）
   - **Branch**：选 `main` 或 `master`
   - **Main file path**：填 `app/Home.py`（注意路径，因为 Home.py 在 app 文件夹里）
4. 点击 `Advanced settings...`
   - **Python version**：选 3.11 或 3.12（不要选 3.13，akshare 不兼容）
   - 其他不用改
5. 点击 `Deploy!`

### 第 4 步：等待部署完成

- 第一次部署大概需要 2-5 分钟
- 你会看到一个黑色的终端窗口在跑安装依赖
- 出现 `You can now view your Streamlit app in your browser.` 就说明成功了
- 页面会自动跳转到你的应用

### 第 5 步：分享链接

- 部署成功后，浏览器地址栏里的链接就是你的公网链接
- 大概长这样：`https://your-username-pharma-intel-platform.streamlit.app`
- 直接把这个链接发给别人就行，任何人都能打开

## 以后怎么更新代码？

### 方法一：GitHub 网页上传（简单）
1. 进入 GitHub 仓库页面
2. 找到要替换的文件，点进去 → 右上角 `Edit`（铅笔图标）
3. 粘贴新内容 → 底部 `Commit changes`
4. Streamlit 会**自动重新部署**，等 1-2 分钟就更新好了

### 方法二：整个文件夹重新上传
1. GitHub 仓库页面 → `Add file` → `Upload files`
2. 拖入新文件 → `Commit changes`
3. 自动重新部署

## 免费版有什么限制？

| 项目 | 限制 |
|------|------|
| 应用数量 | 不限 |
| 访问量 | 每月 10GB 流量（一般够用） |
| 运行时间 | 连续几小时没人访问会自动休眠，有人访问再唤醒（几秒钟） |
| 数据存储 | 不能存数据库（但我们用 JSON 文件，没问题） |

**给老板演示完全够用了。** 如果以后用户多了，可以升级付费版（$10/月起）。

## 常见问题

### Q：部署失败怎么办？
A：看错误信息，常见原因：
1. `requirements.txt` 里的包名写错了
2. Python 版本太高（选 3.11）
3. 文件路径不对（确认 `app/Home.py` 存在）

截图发我，我帮你看。

### Q：能不能用自定义域名？
A：可以，Streamlit 支持绑定自己的域名，在设置里配一下就行。

### Q：数据安全吗？
A：免费版是公开的，任何人有链接就能看。如果涉及敏感数据，有两个方案：
1. 升级 Streamlit 付费版，加密码保护
2. 部署到其他平台（如 Render、Railway、阿里云等）
